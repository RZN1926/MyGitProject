# Инструкция по Markdown

## 1. Заголовки (Headings)
Используйте решетку `#`. Количество решеток — это уровень заголовка:
# Заголовок 1
## Заголовок 2
### Заголовок 3

## 2. Списки (Lists)
- **Маркированный**: используйте дефис `-` или звездочку `*`.
  - Пункт А
  - Пункт Б
1. **Нумерованный**: используйте цифру с точкой `1.`.
  1. Первый шаг
  2. Второй шаг

## 3. Ссылки (Links)
Формат: `[Текст ссылки](URL)`
Пример: [Мой GitHub](https://github.com/RZN1926)
